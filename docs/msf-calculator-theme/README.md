# Calculator form theme CSS (janoga.lv)

Backup copies of **Custom CSS** and **Page layout CSS** for form `#msf-form-3247`.

These live in WordPress under **Multi Step Forms → Banketu piedāvājums → Theme / custom CSS**. They are **not** in the exported step JSON unless you export after saving them.

| File | Paste into |
|------|------------|
| [custom-css.css](custom-css.css) | **Custom CSS** |
| [page-layout-css.css](page-layout-css.css) | **Page layout CSS** |

After pasting: **Update** the form, then hard-refresh `/calculator/`.

**Tip:** Before importing JSON, click **Export JSON** so you keep a backup with `settings.customCss` and `settings.pageCss`.
